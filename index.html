Введите следующий DNS в настройки вашего телефона:

Как изменить DNS на Android:

  1. Откройте "Настройки" → "Подключения" → "Wi‑Fi"
  2. Нажмите на текущую сеть Wi‑Fi
  3. Выберите "Изменить" → "Дополнительно"
  4. В разделе "IP-настройки" выберите "Статический"
  5. Введите DNS в поле "DNS 1"

Как изменить DNS на iOS:

  1. Откройте "Настройки" → "Wi‑Fi"
  2. Нажмите на "i" рядом с текущей сетью
  3. Прокрутите вниз и нажмите "Настроить DNS"
  4. Выберите "Вручную" и добавьте DNS
